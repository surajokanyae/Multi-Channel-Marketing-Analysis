# Multiple Linear Regression – Multi-Channel Marketing Analysis

## Project Overview
This project builds a multivariate Ordinary Least Squares (OLS) regression model to analyze the simultaneous impacts of multiple marketing channels (**TV, Radio, and Social Media**) on corporate **Sales**. Moving beyond simple correlation, this analysis detects multicollinearity, evaluates model stability using Adjusted R-squared, and isolates the precise contribution of each advertising medium while holding alternate channels constant.
